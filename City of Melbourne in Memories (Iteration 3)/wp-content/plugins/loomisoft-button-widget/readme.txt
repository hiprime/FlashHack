== Description ==

Loomisoft's Button Widget plugin provides a widget that allows you to place highly customisable link buttons in your sidebars, footers and other widgetised areas. Designed to be intuitive, the widget provides great flexibility and versatility in defining the button's text, background, borders, paddings and hover characteristics.

= Usage =

You can access the widget in the normal WordPress admin area - Appearance > Widgets. Once you move the widget into the relevant widget area, you can customise the specific button to suit your needs. 

= Features =

Each button provides the following elements that can be managed from within the widget, allowing it to be fully customised:

* Button Text
* Link URL
* Open in New Window
* Button Alignment - options are Full Width (within containing area), Center, Left and Right
* Text Font Family - The font for the button text / includes Google Fonts / leave blank for theme default
* Text Font Size - in pixels (no "px" needed) / leave blank for theme default
* Text Line Height - in pixels (no "px" needed) / leave blank for 1.3em
* Text Color - leave blank for white (for colors you can use the color picker or enter its value in RGB HEX)
* Text Hover Color - leave blank for white
* Text Bold
* Text Hover Bold
* Text Italic
* Text Hover Italic
* Text Underline
* Text Hover Underline
* Background Color - leave blank for light blue
* Background Hover Color - leave blank for dark blue
* Border Width - in pixels (no "px" needed) / leave blank for none
* Border Color - applied when width set / leave blank for dark blue
* Border Hover Color - applied when width set / leave blank for dark blue
* Corner (Border) Radius - in pixels (no "px" needed) / leave blank or 0 for square borders
* Top/Bottom Padding - in pixels (no "px" needed) / leave blank for 10 pixels
* Left/Right Padding - in pixels (no "px" needed) / leave blank for 5 pixels
* Top Margin - in pixels (no "px" needed) / leave blank for 10 pixels
* Bottom Margin - in pixels (no "px" needed) / leave blank for 10 pixels
* Left/Right Margin - in pixels (no "px" needed) / leave blank for 0 pixels

= Notes =
* "Padding" is the space between the text and the border/edge of the button.
* "Margin" is the space between the button and elements above/below and the edge of the containing element either side. Do be aware that elements and below may also have margins set. The top and bottom margins may be set to negative values, but do be aware that the website's theme may have defined containing blocks with "overflow" hidden.

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/loomisoft-button-widget' directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Once installed, usage information may be found in the WordPress admin menu item "Button Widget"

== Frequently Asked Questions ==

= Are buttons easy to add and edit? =

Yes. They can be added into any of the sidebars in the normal WordPress "Appearance > Widgets" Area. Once added, each button can be customised for your needs.

= Can buttons be placed within the body of a page/post or other areas? =

No. They are currently only designed to be placed in sidebars and other widgetised areas.

= Are there additional features planned in future releases? =

Aspects that may be added in future releases include:

* Having icons with the buttons
* Color gradient for backgrounds
* Outer/drop shadow or glow around the button
